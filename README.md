## 📌 프로젝트 소개

공공 API를 활용해 전국 전기차 충전소 데이터를 실시간으로 관리하고,  
사용자가 현재 위치 기반으로 충전소를 탐색·예약할 수 있는 서비스입니다.

## 👤 본인 담당 기능

### 🗺 지도 기능

**스파이더파이어 (Spiderfire) — 직접 구현**
- 동일 위치에 마커가 겹칠 때 외부 라이브러리 없이 삼각함수(`Math.cos` / `Math.sin`)로 원형 분산 알고리즘 직접 설계
- 마커 수에 따라 반경 동적 조절, 360도를 균등 분할하여 배치

**마커 클러스터링**
- `MarkerClusterer` 도입으로 수천 개 마커 렌더링 시 DOM 생성 최소화
- 클러스터 상태에서도 목록 호버 강조 연동 (끊김 없는 UX 유지)

**충전소 목록 & 검색**
- 현재 위치 기준 반경 1.5km 내 충전소 100개 거리순 조회
- 급속·완속 충전기 구분 표시 및 실시간 거리 정보 제공
- 반경 내 충전소만 검색되도록 필터링, 10분 주기 실시간 현황 갱신

**마커 색상 & 인터랙티브 강조**
- 충전 상태(여유·보통·혼잡·고장 등 6가지)에 따른 마커 색상 시각화
- 목록 호버 시 해당 마커 확대 + 상세 페이지 연동 파란색 강조

---

### ⚙️ 백엔드 & 쿼리 최적화

**SQL 쿼리 최적화**
- 상관 서브쿼리를 `GROUP BY` 인라인 뷰 조인으로 통합 → CHARGER 테이블 스캔 **3회 → 1회** 감소
- 위경도 `BETWEEN` 사각형 필터 추가로 인덱스 스캔 유도, 응답 성능 개선

**공공 API 스케줄러 최적화**
- 전체 반복 호출 구조 → 8천 개 단위 배치 처리로 개선
- 최근 10분간 변경된 데이터만 갱신하는 구조로 API 트래픽 및 처리 시간 최적화
